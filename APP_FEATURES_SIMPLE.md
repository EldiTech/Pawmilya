# Pawmilya Native Simple Feature List

This app has these main features:

1. Guest browsing
- View pets and basic app pages without logging in.

2. Login and account system
- Sign up, login, logout, and account switching.
- Two-factor authentication support.

3. Role-based access
- Different experiences for User, Shelter, Rescuer, and Admin.

4. Pet adoption features
- Users can browse pets and submit adoption requests.
- Shelters can approve, reject, dispatch, handover, and complete adoption flow.
- Adoption status tracking from start to finish.

5. Adoption chat
- User and shelter can chat inside adoption requests.

6. Shelter management features
- Shelter profile setup.
- Shelter pet listing and rescued pet tracking.
- Shelter adoption request management.
- Shelter transfer requests and shelter chats.
- Shelter funds screen.

7. Rescue features
- Users can report rescue needs.
- Rescuers can register and manage rescue missions.
- Rescue mission status updates.

8. Admin features
- Admin dashboard and settings.
- Manage users and pets.
- Review shelter and rescuer applications.
- Review rescue reports and transfers.

9. Notifications
- Users can receive and manage notifications.

10. AI chatbot assistant
- Jemoy AI assistant is available from user settings.

11. Profile and settings
- Edit profile, avatar, password, and account preferences.

12. Firebase-backed data
- Uses Firebase Auth and Firestore for authentication and app data.
